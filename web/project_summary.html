04 Conclusions Business

Ce que les données révèlent sur Olist

01 Le délai de livraison est le principal prédicteur de satisfaction. Pearson r = −0.33 (p < 0.05). Tous les modèles (RF, GB, XGB, DT) confirment que delivery_days et delivery_delay sont les features les plus importantes. Livrer avant la date estimée améliore directement la satisfaction.
02 La satisfaction peut être prédite avant le contact client. Le meilleur modèle (Gradient Boosting, AUC=0.683) détecte correctement les insatisfaits sur la base de données logistiques disponibles au moment de la livraison.
03 São Paulo est le hub logistique le plus performant. Frais de livraison les plus bas, proportion de livraisons rapides la plus élevée (37.4% Fast). Les vendeurs SP bénéficient d'un avantage structurel.
04 3 segments clients distincts identifiés (RFM). Cluster 0 : clients perdus (Recency 389j). Cluster 1 : gros dépensiers inactifs → risque de départ. Cluster 2 : clients récents et fidèles → cœur de cible.
05 Les associations entre catégories sont rares mais exploitables. home_confort → bed_bath_table (lift=3.85, confiance 57%). Recommandation cross-selling directement actionnable.
